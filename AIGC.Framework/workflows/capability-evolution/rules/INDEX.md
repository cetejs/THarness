# AIGC 能力演化工作流规则索引

默认只读本索引。按命中条件读取具体规则。

| --- | --- |
| `scope.md` | 判断是否应进入能力演化工作流，以及本轮边界。 |
| `source-intake.md` | 需要登记外部 harness、旧项目、文档或工具来源。 |
| `comparison.md` | 需要对比外部能力与当前 AIGC 能力。 |
| `extraction.md` | 需要提取可复用能力、规则、模板或知识。 |
| `versioning.md` | 需要更新 AIGC 能力索引、版本或变更记录。 |
| `wiki-sync.md` | 需要把可复用结论写入通用 wiki。 |
| `verification.md` | 需要判断本轮能力演化是否完成。 |
