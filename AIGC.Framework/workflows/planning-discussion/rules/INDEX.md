# 策划案讨论工作流规则索引

默认只读本索引。按命中条件读取具体规则。

| 文件 | read_when |
| --- | --- |
| `scope.md` | 判断工作流边界、非目标、是否应该启动。 |
| `questioning.md` | 需求存在歧义、缺少约束、需要用户确认。 |
| `game-design-methodology.md` | 用户讨论游戏玩法、战斗体验、系统机制、玩家理解或原型验证。 |
| `decision.md` | 用户确认方案、产生决策、需要记录取舍。 |
| `project-adaptation.md` | 需要在目标项目建立运行记录、项目 wiki 或项目文档规则。 |
| `wiki-sync.md` | 需要沉淀通用知识或项目知识。 |
| `verification.md` | 需要判断本轮策划案是否完成。 |
