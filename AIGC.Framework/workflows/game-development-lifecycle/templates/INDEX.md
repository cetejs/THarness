# 游戏开发生命周期模板索引

| 模板 | 用途 |
| --- | --- |
| `lifecycle-plan.md` | 建立一轮游戏开发生命周期计划。 |
| `development-readiness-checklist.md` | 记录进入实现前的工具链、界面、素材、数据和验证硬门禁。 |
| `screen-contract.md` | 记录玩家可见界面的布局、按钮、数据和跳转。 |
| `asset-pipeline-contract.md` | 记录素材从来源、生成、导入到运行时引用的管线合同。 |
| `runtime-debug-record.md` | 记录运行时问题、日志、复现、修复和回归。 |
| `verification-retro.md` | 记录交付验证和复盘沉淀。 |
| `project-execution-sync-check.md` | 记录目标项目适配层的工作流、状态、验证入口、任务和追溯同步检查。 |

模板只提供格式，不保存具体项目内容。
