## 卡面根因

旧卡面具备边框和基本信息，但看起来普通，主要原因是：

1. 卡牌身份主要靠边框颜色，类型差异不够强。
2. 卡面中部有黑色信息块，像遮挡或占位。
3. 插画区没有按卡牌类型形成视觉差异。
4. 中部重复显示卡名，没有形成规则题签层级。
5. 小卡使用全局字体，顶部卡名和底部类型容易被裁剪。
