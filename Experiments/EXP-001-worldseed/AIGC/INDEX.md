# EXP-001-worldseed AIGC 入口

本目录保存实验一 WorldSeed 项目的 AIGC 适配层。后续处理实验一项目事实、开发任务、契约、运行记录和验证记录时，必须先从本文件进入项目上下文。

## 默认读取

1. `ADAPTER.md`
2. `wiki/INDEX.md`

## 读取路由

| 需求 | 读取入口 |
| --- | --- |
| 了解项目目标和范围 | `wiki/project.md` |
| 了解当前状态和可执行入口 | `wiki/status.md` |
| 查看项目架构索引 | `wiki/architecture/INDEX.md` |
| 查看设计文档索引 | `wiki/design/INDEX.md` |
| 查看开发任务、子任务和接口契约入口 | `wiki/development/INDEX.md` |
| 查看实验规划、实验规则和实验结论 | `wiki/experiments/INDEX.md` |
| 查看新增内容、验证和来源追溯 | `wiki/traceability/INDEX.md` |
| 查看构建、测试、运行和外部依赖 | `wiki/source-map/INDEX.md` |
| 查看文本编码和读取规则 | `wiki/rules/encoding.md` |
| 查看已确认或废弃决策 | `wiki/decisions/INDEX.md` |
| 查看项目内工作流约束 | `wiki/workflows/INDEX.md` |
| 查看未确认问题 | `wiki/open-questions.md` |
| 恢复最近任务上下文 | `workflows/{workflow_id}/runs/INDEX.md` |

## 禁止

- 禁止为了保险读取全部历史运行记录。
- 禁止把未确认想法写入正式设计或架构页面。
- 禁止把单次实验结果直接写成稳定项目事实。
- 禁止新增正式项目内容但不登记追溯记录。
- 禁止把项目事实写入 `AIGC.Framework`。
- 禁止把实验一项目事实写入其他实验目录。
