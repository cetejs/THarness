## 结论

所有可动态调整的生成参数、路径、模型、导出选项和玩法数值必须进入配置或 GameSpec，不散落在代码里。

## 初始配置类型

| 类型 | 示例 |
| --- | --- |
| 项目路径 | Godot 工程路径、输出路径、缓存路径。 |
| 生成策略 | 是否允许占位资源、目标平台、模板版本。 |
| 模型配置 | 对话模型、图像模型、音频模型。 |
| 游戏数值 | 生命、伤害、速度、冷却、奖励。 |
| 验证配置 | Godot 可执行路径、截图目录、测试场景。 |

## 验证

调整配置不应需要修改业务代码。
