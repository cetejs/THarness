# 开发入口

开发前必须先建立任务页。没有任务页，不进入代码实现。

| 目录 | read_when |
| --- | --- |
| `tasks/INDEX.md` | 需要查看主开发任务。 |
| `subtasks/INDEX.md` | 需要查看角色子任务。 |
| `contracts/INDEX.md` | 需要查看跨模块接口契约。 |

## 任务规则

- 主任务写入 `tasks/{task_id}.md`。
- 子任务写入 `subtasks/{role}-{subtask_id}.md`。
- 接口契约写入 `contracts/{contract_id}.md`。
- 每个任务必须包含目标、范围、禁止范围、成功标准和验证方式。
