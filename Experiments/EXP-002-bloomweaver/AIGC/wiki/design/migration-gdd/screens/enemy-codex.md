# 敌人图鉴界面迁移策划案

## 定位

敌人图鉴用于展示敌潮单位资料、五行属性、行为、克制关系、出现波次和掉落。首版可展示普通、精英、Boss 三类基础敌人。

## 基本信息

| 项 | 内容 |
| --- | --- |
| screen key | `enemy-codex-screen` |
| 参考图 | `images/ui-reference-20260517/enemy-codex.png` |
| 原 Unity Prefab | `Assets/Prefabs/UI/Screens/enemy-codex-screen.prefab` |
| 目标技术 | 新引擎 UI 系统 |
| 显示层级 | 主页面或覆盖层 |
| 来源阶段 | 花灵图鉴、主界面扩展入口 |
| 退出方向 | 来源界面、花灵图鉴、提示框 |

## 玩家目标

- 查看已遭遇敌人。
- 理解敌人五行和行为。
- 查看推荐应对策略。
- 切回花灵图鉴。

## 视觉与布局

| 区域 | 内容 |
| --- | --- |
| 左侧 | 敌人列表。 |
| 中央 | 选中敌人立绘。 |
| 右侧 | 基础、行为、克制、波次、掉落标签。 |
| 底部 | 返回、花灵图鉴、查看应对策略。 |

## 数据需求

| 字段 | 类型 | 来源 | 用途 |
| --- | --- | --- | --- |
| `enemyEntries` | list | 图鉴/敌人配置 | 敌人列表。 |
| `selectedEnemyId` | string | UI 状态 | 当前敌人。 |
| `selectedTab` | enum | UI 状态 | 当前标签。 |
| `encounterState` | object | 存档系统 | 是否遭遇。 |
| `enemyPortrait` | asset | 资源系统 | 立绘或剪影。 |
| `strategyText` | string | 策略配置 | 应对建议。 |

## 交互规格

| 控件 | 操作 | 结果 |
| --- | --- | --- |
| 敌人列表项 | 点击 | 切换选中敌人。 |
| 标签按钮 | 点击 | 切换详情。 |
| 花灵图鉴 | 点击 | 打开花灵图鉴。 |
| 查看应对策略 | 点击 | 显示策略提示或展开策略面板。 |
| 返回 | 点击 | 回来源界面。 |

## 无效输入

- 未遭遇敌人显示剪影和基础提示。
- 没有策略数据时策略按钮禁用。
- 资源缺失时使用通用敌人占位图。

## 迁移实现要点

- 敌人图鉴不依赖当前战斗中的敌人实例。
- 遭遇状态和击杀统计来自存档或局内统计汇总。
- 策略内容应由配置提供，方便后续平衡调整。

## 迁移验收

- 点击敌人列表能刷新立绘和信息。
- 未遭遇敌人不会泄露完整资料。
- 花灵图鉴切换入口可用。
- 策略按钮在无数据时有明确禁用状态。
