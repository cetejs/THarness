# 花灵图鉴界面迁移策划案

## 定位

花灵图鉴用于展示花灵资料、五行、职业、扫描形状、技能定位和背景故事。首版可以只展示当前 5 名花灵。

## 基本信息

| 项 | 内容 |
| --- | --- |
| screen key | `codex-screen` |
| 参考图 | `images/ui-reference-20260517/codex.png` |
| 原 Unity Prefab | `Assets/Prefabs/UI/Screens/codex-screen.prefab` |
| 目标技术 | 新引擎 UI 系统 |
| 显示层级 | 主页面或覆盖层 |
| 来源阶段 | 主界面、暂停、胜利结算 |
| 退出方向 | 来源界面、敌人图鉴、局外养成 |

## 玩家目标

- 查看已解锁花灵。
- 理解花灵职业和五行定位。
- 查看扫描形状和技能说明。
- 切换到敌人图鉴或局外养成。

## 视觉与布局

| 区域 | 内容 |
| --- | --- |
| 左侧 | 花灵列表。 |
| 中央 | 选中花灵立绘。 |
| 右侧 | 基础信息、技能、扫描、灵脉、故事标签。 |
| 底部 | 返回、敌潮图鉴、万花残庭。 |

## 数据需求

| 字段 | 类型 | 来源 | 用途 |
| --- | --- | --- | --- |
| `flowerEntries` | list | 图鉴系统/花灵配置 | 左侧列表。 |
| `selectedFlowerId` | string | UI 状态 | 当前花灵。 |
| `selectedTab` | enum | UI 状态 | 当前标签。 |
| `unlockState` | object | 存档系统 | 是否解锁。 |
| `flowerPortrait` | asset | 资源系统 | 立绘。 |
| `scanShape` | object | 花灵配置 | 扫描显示。 |

## 交互规格

| 控件 | 操作 | 结果 |
| --- | --- | --- |
| 花灵列表项 | 点击 | 切换选中花灵。 |
| 标签按钮 | 点击 | 切换右侧详情。 |
| 敌潮图鉴 | 点击 | 打开敌人图鉴。 |
| 万花残庭 | 点击 | 打开局外养成或提示未开放。 |
| 返回 | 点击 | 回来源界面。 |

## 无效输入

- 未解锁花灵只显示允许公开的信息。
- 立绘资源缺失时显示剪影。
- 局外养成未开放时按钮禁用或提示。

## 迁移实现要点

- 图鉴数据应从配置和存档合成，不从战斗运行时对象读取。
- 扫描形状要用可视化格子展示，帮助玩家理解布阵。
- 花灵故事不影响战斗规则，保持纯展示。

## 迁移验收

- 点击花灵列表能刷新立绘和信息。
- 标签切换内容正确。
- 未解锁花灵显示锁定状态。
- 返回后回到正确来源界面。
