## 9. 验收合同

| 验收项 | 方式 | 通过标准 |
| --- | --- | --- |
| 效果图一致性 | 同视口截图对比 | 关键区域位置、比例、层级和色彩符合允许偏差。 |
| 真实输入 | 鼠标或触摸点击关键控件 | 点击范围、反馈、跳转和禁用态符合控件合同。 |
| 缩放适配 | 默认视口和至少一个缩放视口截图 | 不裁切关键区域，文本不溢出，控件不重叠。 |
| 动态数据 | 替换长文本、空数据和不同数值 | 按数据合同显示。 |
| 回归 | 重复进入、退出、返回和快速点击 | 无重复触发、无残留覆盖层、无错误跳转。 |
