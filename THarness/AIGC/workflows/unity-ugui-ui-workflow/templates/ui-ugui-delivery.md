## 10. 验收标准

1. Unity 编译通过。
2. Editor 菜单能创建 Prefab。
3. Prefab 中所有 `[SerializeField]` 字段已绑定。
4. 所有 Sprite 路径存在或缺失日志清晰。
5. 所有旧版 Text 文本可正常显示。
6. 所有 Button 点击事件只抛出事件，不写业务跳转。
7. Unity 截图已回传并完成至少一轮校准。
8. 施工资源只来自 UI 开发者审核通过的正式资源集合。
9. 未审核元素资源、污染元素、未切素材合集、ReferenceOnly 和完整展示图没有进入施工依赖。
