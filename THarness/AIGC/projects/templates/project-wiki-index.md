# 项目 Wiki 入口模板

## 读取路由

默认只读本索引。按任务意图命中 `read_when` 后，再读取对应文件。

| 文件 | read_when |
| --- | --- |
| `README.md` | 需要理解项目 wiki 定位、边界和工作方式。 |
| `SCHEMA.md` | 需要创建、修改或审查项目 wiki 页面格式。 |
| `project.md` | 需要了解项目名称、目标、范围和当前适配配置摘要。 |
| `status.md` | 需要了解当前项目状态或可执行入口。 |
| `architecture/INDEX.md` | 需要检索项目入口、目录结构、模块边界、依赖方向或配置入口。 |
| `source-map/INDEX.md` | 需要追溯重要文件、构建测试命令或外部服务。 |
| `design/INDEX.md` | 需要检索策划案、功能设计、核心体验、成功标准或待确认设计问题。 |
| `development/INDEX.md` | 需要检索主程评审、开发任务拆解、子任务分配、接口契约或验收入口。 |
| `decisions/INDEX.md` | 需要查看已确认决策。 |
| `workflows/INDEX.md` | 需要查看项目内开发或验证流程。 |
| `open-questions.md` | 需要确认未决问题或阻塞。 |
| `rules/retrieval.md` | 需要降低项目 wiki 检索 token。 |
| `rules/building.md` | 需要新增项目 wiki 页面。 |
| `rules/health.md` | 需要检查项目 wiki 健康状态。 |
